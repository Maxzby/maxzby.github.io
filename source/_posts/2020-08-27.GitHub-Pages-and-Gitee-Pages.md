## 网站速度访问

### 国内

``` bash
国内推荐使用 "https://maxzby.gitee.io"访问
国内使用Gitee Pages提供服务，访问效果极佳
但是国内不是每天同步，（毕竟两个不一样的服务器，想较快看文章优先海外
```

国内优点：
访问速度快 免服务器 稳定
缺点：
同步文章速度慢（手动同步

### 海外

``` bash
海外推荐使用 "https://maxzby.github.io" 访问
海外使用Github Pages提供服务，表现一般般
秒同步文章
```

海外优点：
秒同步文章 免服务器
缺点：
访问速度一般般 不稳定
